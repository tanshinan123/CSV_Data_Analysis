CSV_Data_Analysis项目描述：
智能数据分析助手是一款面向业务人员的AIGC数据洞察工具。用户上传CSV后即可通过自然语言提出分析诉求，系统调用GPT系列及头部模型理解意图，并联动Pandas自动完成数据查询、清洗与图表可视化（条形图/折线图/散点图）。结果以结构化JSON返回，适用于运营分析、市场研究、报表生成等场景，大幅降低数据分析门槛。

项目概述：
基于LangChain和Pandas的智能数据分析工具，通过自然语言交互实现CSV数据的自动分析和可视化，让非技术用户也能轻松进行复杂的数据分析。

技术架构：
- 前端框架：Streamlit - 提供数据上传和结果展示界面
- AI框架：LangChain - 构建数据分析智能体
- 数据处理：Pandas - 核心数据分析引擎
- 可视化：Streamlit Charts - 支持多种图表类型
- 大语言模型：GPT-4-turbo - 理解用户查询并生成分析代码

核心功能：
- 智能数据理解：自动解析CSV文件结构和数据类型
- 自然语言查询：用户用自然语言描述分析需求
- 多格式输出：
   - 文字描述分析结果
   - 数据表格提取
   - 条形图、折线图、散点图可视化
- 结构化输出：通过JSON格式确保输出的一致性和可解析性

技术亮点：
- 使用LangChain的create_pandas_dataframe_agent创建专业数据分析智能体
- 自定义Prompt模板实现结构化输出控制
- 支持多种图表类型的自动选择
- 错误处理机制确保系统稳定性

应用场景：
- 业务数据分析
- 市场调研报告
- 财务数据可视化
- 用户行为分析
- 运营数据监控

项目价值：
将复杂的数据分析过程简化为自然语言交互，体现了AIGC在降低技术门槛、提升工作效率方面的巨大潜力。

项目描述：
文档智能问答是一款基于AIGC与RAG检索增强技术的知识获取工具。用户上传PDF后，系统将文档向量化入库，并结合GPT系列及头部模型进行检索与生成，对问题进行基于文档证据的精准回答。支持对话式追问与上下文记忆，适用于技术手册查询、法律合规、学术资料研读及企业知识库检索等场景。

PDF_QA_tool项目概述：
基于RAG（检索增强生成）技术的智能PDF问答系统，能够理解PDF文档内容并回答用户相关问题，实现文档知识的智能提取和交互式查询。

技术架构：
- 前端框架：Streamlit - 提供文件上传和问答界面
- AI框架：LangChain - 构建RAG对话链
- 文档处理：PyPDFLoader - PDF文档解析
- 文本分割：RecursiveCharacterTextSplitter - 智能文本分块
- 向量化：OpenAI Embeddings - 文本向量化处理
- 向量存储：FAISS - 高效向量数据库
- 检索增强：ConversationalRetrievalChain - 结合检索和生成的对话链

核心功能：
- 文档解析：自动提取PDF文本内容并进行智能分块
- 向量化存储：将文档内容转换为向量并建立索引
- 智能检索：根据用户问题检索相关文档片段
- 上下文问答：结合检索内容和对话历史生成准确回答
- 对话记忆：保持多轮对话的上下文连续性

技术亮点：
- 采用RAG架构，确保回答基于实际文档内容
- 使用FAISS向量数据库实现高效相似性搜索
- 智能文本分割策略，平衡上下文长度和检索精度
- 支持多轮对话，具备上下文理解能力

应用场景：
- 法律文档问答
- 技术手册查询
- 学术论文分析
- 企业知识库问答
- 培训材料学习辅助

项目价值：
展示了RAG技术在文档智能问答领域的应用，体现了对知识管理、信息检索和自然语言处理技术的综合运用，为构建企业级知识问答系统提供了完整的技术方案。
