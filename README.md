项目描述：
智能数据分析助手是一款面向业务人员的AIGC数据洞察工具。用户上传CSV后即可通过自然语言提出分析诉求，系统调用GPT系列及头部模型理解意图，并联动Pandas自动完成数据查询、清洗与图表可视化（条形图/折线图/散点图）。结果以结构化JSON返回，适用于运营分析、市场研究、报表生成等场景，大幅降低数据分析门槛。

项目概述：
基于LangChain和Pandas的智能数据分析工具，通过自然语言交互实现CSV数据的自动分析和可视化，让非技术用户也能轻松进行复杂的数据分析。

技术架构：
- 前端框架：Streamlit - 提供数据上传和结果展示界面
- AI框架：LangChain - 构建数据分析智能体
- 数据处理：Pandas - 核心数据分析引擎
- 可视化：Streamlit Charts - 支持多种图表类型
- 大语言模型：GPT-4-turbo - 理解用户查询并生成分析代码

核心功能：
- 智能数据理解：自动解析CSV文件结构和数据类型
- 自然语言查询：用户用自然语言描述分析需求
- 多格式输出：
   - 文字描述分析结果
   - 数据表格提取
   - 条形图、折线图、散点图可视化
- 结构化输出：通过JSON格式确保输出的一致性和可解析性

技术亮点：
- 使用LangChain的create_pandas_dataframe_agent创建专业数据分析智能体
- 自定义Prompt模板实现结构化输出控制
- 支持多种图表类型的自动选择
- 错误处理机制确保系统稳定性

应用场景：
- 业务数据分析
- 市场调研报告
- 财务数据可视化
- 用户行为分析
- 运营数据监控

项目价值：
将复杂的数据分析过程简化为自然语言交互，体现了AIGC在降低技术门槛、提升工作效率方面的巨大潜力。
